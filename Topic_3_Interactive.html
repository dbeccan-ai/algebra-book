FUNCTIONS & GRAPHING SUCCESS STRATEGIES

For Students:

  • Master function notation - understand what f(x), f(a), and f(g(x)) represent
  • Practice transformations systematically - know how each parameter affects the graph
  • Use the vertical line test - to determine if a relation is a function
  • Identify key features - intercepts, vertex, asymptotes, domain, and range
  • Connect algebraic and graphical representations - see the relationship between equations and graphs
  • Check your work graphically - use the TI-84 to verify algebraic solutions

Common Student Errors to Avoid:

  • Confusing domain and range - remember domain is input (x), range is output (y)
  • Incorrect transformation direction - (x - h) shifts RIGHT h units, not left
  • Function notation mistakes - f(x + 2) ‚â† f(x) + 2
  • Composition order errors - (f ‚àò g)(x) = f(g(x)), work from inside out
  • Asymptote identification errors - vertical at zeros of denominator, horizontal by degree comparison
  • Forgetting domain restrictions - especially for rational, radical, and logarithmic functions

Time Management Tips:

  • Function evaluation: 2-3 minutes per problem

  • Strategy: Always graph when in doubt - visual confirmation helps catch errors

Verification Strategies:

  1. Vertical Line Test: Verify function by checking if any vertical line intersects graph more than once
  2. Input-Output Check: Substitute values to confirm function rule produces correct outputs
  3. Domain Verification: Check that all x-values in domain produce valid y-values
  4. Graph Matching: Compare calculated features (intercepts, vertex) with graph
  5. Symmetry Testing: Verify even/odd function properties algebraically
  6. Calculator Graphing: Use TI-84 to confirm hand-drawn graphs

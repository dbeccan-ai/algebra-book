  2. Substitute Values: Test both original and factored forms with x = 0, 1, or 2

  4. Leading Coefficient: Verify sign and value match the original
  5. Prime Check: Confirm when no further factoring is possible
  6. Calculator Verification: Use table or graph features to compare expressions
